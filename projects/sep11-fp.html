Freedom Project (SEP11)

Context

For SEP11 the Freedom Project is making whatever we want functional through Javascript which involves using a JS library, framework, or some other tool to accomplish that goal

Content

My project was on making a fast platformer through Javascript which in my case I had also used a game engine called MelonJS which had helped me use tools like Tiled to make the level but also within itself it had functions which made certain things be made with ease.

Reflection

Throughout working on this project I had improved on things I had to do last year which was learning on my own and time management. Alongside reinforcing things I had learned from the previous year, I had learned more with Javascript and how it worked as it gave me the exposure for objects and classes which would be more prevelant in APCSA but also game design with how to make a proper game level.
