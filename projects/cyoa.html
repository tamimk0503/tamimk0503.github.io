Create Your Own Adventure (CYOA)

Context

This project is about using the knowdelge of using a command line alongside with git and github to make an interactive adventure where you can make your own choices

Content

The theme we had decided on was breaking out of jail and from it we had made two major sides which were being social or being kept to yourself which shows a huge difference at the end

Reflection

The challenge we faced was creating the choices as they had to make sense and have enough depth to it. The next step I would consider if I had to return back to this project is possibly adding a new route and adding better images.
